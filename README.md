# Java Web Development : Movie Review System

####**本示例是使用JavaWeb技术实现一个简单的文章管理系统（新闻管理系统）其中主要功能如下：**

- **用户和管理员登录**
- **用户发布新文章、文章详情查看、文章修改、文章删除与恢复**
- **用户查看他人对自己授权的文章及其文章信息**
- **用户将自己的文章对他人进行授权**
- **管理员对普通用户新发布的文章进行审核和删除**
- **管理员查看普通用户发布的所有文章及其详情**
- **管理员发布新文章**
